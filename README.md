# Software Quality Assurance (SQA) Practice Lab 🧪

Welcome to my SQA practice repository! This project serves as a centralized hub for all my learning, exercises, and projects completed during my **Software Quality Assurance** course

## 🚀 Purpose
The goal of this repository is to document my journey in mastering software testing, moving from manual testing fundamentals to automated quality engineering.

## 📂 What's Inside?
- **Manual Testing:** Bug reports, Test Cases, and Test Plans for various web applications.
- **Automation Scripts:** Practice scripts using tools like Selenium, Playwright, or Cypress.
- **API Testing:** Postman collections and environment setups.
- **Performance Testing:** Basic JMeter scripts and results.
- **Documentation:** SDLC/STLC notes and best practices.

## 🛠 Tools & Technologies
- **Management:** Jira, Trello
- **Testing Tools:** Selenium WebDriver, Postman, JMeter
- **Version Control:** Git & GitHub
- **Documentation:** Google Sheets / MS Excel for Test Cases
